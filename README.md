# Octree-OBB-Grasp-Planner
Efficient path planning method based on three-dimensional point cloud octree-oriented bounding Box map construction for robotic grasping
## Notice on Code and Dataset Availability
Thank you for your interest in this project, Efficient Path Planning Method Based on Three-Dimensional Point Cloud Octree-Oriented Bounding Box Map Construction for Robotic Grasping. To ensure compliance with institutional policies and confidentiality requirements, the source code and associated datasets for this work are not publicly available at this moment.

In line with our commitment to open science and reproducibility, we plan to release the full codebase and datasets after the official publication of the corresponding research article. Additionally, a demonstration video will be provided to illustrate the proposed method, including its implementation and deployment in practical robotic grasping scenarios, allowing viewers to gain a clear and intuitive understanding of the approach and its performance.

We sincerely appreciate your understanding and patience.
